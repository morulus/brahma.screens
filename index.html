Brahma.Screens

SCREENS API

Easy to use

It's realy simple to create.
<div id="gallery">
	<img src="photo1.jpg" data-screens-x="0" data-screens-y="0" />
	<img src="photo2.jpg" data-screens-x="1" data-screens-y="0" />
	<img src="photo3.jpg" data-screens-x="2" data-screens-y="0" />
	<div data-screens-x="3" data-screens-y="0">Last slide<div  />
</div>
<script>
	Brahma("#gallery").app("screens");
</script>
